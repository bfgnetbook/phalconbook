# Test application based on the book 'WEB DEVELOPMENT WITH PHALCON PHP'.

Skeleton application using  [Phalcon framework](https://phalcon.io).

## Requirements

- PHP >= 7.4.1
- Phalcon >= 5.0.0

## Installing via Git

```bash
cd my-project
git clone --depth=1 https://github.com/bfgnetbook/phalconbook .
composer install

```
